# Politique de confidentialite — Epitech Intra Lyceens

> Derniere mise a jour : 2026-04-06

## Responsable du traitement

Epitech — [adresse du campus]
Contact : dpo@epitech.eu

## Donnees collectees

Dans le cadre de l'organisation des activites pour lyceens (stages, evenements, coding clubs), nous collectons :

- **Nom et prenom** : pour vous identifier
- **Adresse email** : pour vous contacter et vous connecter
- **Etablissement scolaire** : pour organiser les activites
- **Niveau scolaire** : pour adapter le contenu
- **Notes et progression** : pour suivre votre parcours pedagogique

## Finalite du traitement

Vos donnees sont utilisees exclusivement pour :
1. Gerer votre inscription aux activites Epitech
2. Suivre votre progression pedagogique
3. Vous contacter pour les informations liees aux activites

## Base legale

- Execution du contrat d'inscription pour les donnees obligatoires
- Consentement pour les donnees facultatives

## Destinataires

Vos donnees sont accessibles uniquement :
- Au personnel Epitech de votre campus
- A votre referent d'activite

Aucune donnee n'est transmise a des tiers commerciaux.

## Duree de conservation

Vos donnees sont conservees pendant la duree de votre activite, puis 1 an apres la fin de celle-ci. Elles sont ensuite supprimees.

## Vos droits

Conformement au RGPD, vous disposez des droits suivants :

| Droit | Description |
|---|---|
| **Acces** | Obtenir une copie de vos donnees |
| **Rectification** | Corriger des donnees inexactes |
| **Effacement** | Demander la suppression de vos donnees |
| **Portabilite** | Recevoir vos donnees dans un format standard |
| **Opposition** | Vous opposer au traitement de vos donnees |

Pour exercer ces droits, contactez : **dpo@epitech.eu**
Delai de reponse : 1 mois maximum.

## Mineurs

Si vous avez moins de 15 ans, le consentement de votre representant legal est requis.

## Securite

Nous mettons en oeuvre des mesures techniques pour proteger vos donnees :
- Acces protege par authentification
- Chiffrement des communications (HTTPS)
- Controle d'acces par role
- Surveillance et audit regulier
